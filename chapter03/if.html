제어문

프로그램의 실행 순서 (=흐름)을 제어하는 문장(=문법, 구문)이다.

제어문은 크게 1)조건문, 선택문, 반복문으로 구분한다.

조건문

조건식의 값이 true,false 에 따라 각가 다른 코드를 실행하는 문법

특정조건에 의해서 프로그램 실행이 분기에 따라 나눠지는것

  1. 단순 if
  2. if - else
  3. if - else if
  4. 중첩if

    5. 단순 if

    조건식이 true 일때만 실행되는 구문, false 일때는 실행 x

                if(조건식){
                ....자바스크립트 코드들....;
            }

    if - else

    단순 if 는 조건식이 true 일때만 실행되지만, if - else 는 true 일때와 false 일때 실행하는 구문이 따로 존재함

                if(조건식){
                ..true 일때 실행코드..;
            }else{
                ..false 일때 실행코드..;
            }
