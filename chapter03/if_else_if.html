if~ else if

두가지 이상의 조건식과 정해놓은 조건을 만족하지 않았을 때 실행되는 코드로 이뤄져 있습니다. 

            if(조건식1){
            조건식 1이 true 면 실행될 코드1;
        } else if(조건식2){
            조건식 2가 true면 실행될 코드2;
        } else if(조건식 3,4,5,....){
            조건식 3 4 5 가 true 면 실행될 코드
        } else{
            조건식 5까지 일치하는게 없으면 실행될 코드
        }
