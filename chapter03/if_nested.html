중첩 if

if 구문 안에 또 if 구문을 작성하는 규칙, 너무 많이 중첩 () 

           if(조건식){
               if(다른 조건식2){
                  코드
               }
           }

중첩 if 예시

질의 응답창을 이용해서,로그인 하는 프로그램을 작성할것

미리 정해진 id/pw와 사용자가 입력한 id/pw 를 비교해서,

id가 일치하면 pw를 비교해서 그것마저 일치하면 로그인 성공하고 id가 일치하나 pw를 비교해서 일치하지 않으면 로그인 실패, 또는 id마저 미리 정해진 id와 다르다면,... 의 일치 유무에 따라 성공, 실패에 따른 화면(view)를 제공하는것
